### Проект спринта: калькулятор денег и калорий

Создание калькулятора денег и калорий с использованием Python. Кроме того, данный скрипт должен проходить проверку существующими тестами pytest, а всю работу над скриптом следует проводить в развёрнутом виртуальном окружении.

**Умеет считать**:
- Сколько можно ещё потратить ₽/$/€ на день запуска (get_today_cash_remained)
- Сколько ещё можно получить каллорий (get_calories_remained)
- Сколько было потрачено сегодня (get_today_stats)
- Сколько каллорий получено/потрачено денег за последнюю неделю (get_week_stats)

## Настройка проекта
Перед использованием требуется дописать проект, иначе при запуске ничего не будет выведено. Для этого стоит учитывать, что необходимо передать в класс CashCalculator бюджет или максимально возможное потребление каллорий за один день, в числовом формате (1000)

После требуется отправить данные. Для этого необходимо использовать add_record(Record(amount=x, comment='y', data="z") где:
- x — количество потраченных денег / съеденных каллорий
- y — комментарий к потраченным деньгам / описание съеденного
- z — дата в формате День.Число Месяца.Год Например: 08.11.2019

Возможна отправка данных без поля даты, в таком случае будет использована дата на день отправки. Пример:

    add_record(Record(amount=x, comment='y')

В дальнейшем, можно запустить любую необходимую функцию, из упомянутых выше. Функция get_today_cash_remained умеет конвертировать записанные расходы в валюту, поэтому нужно использовать usd, eur или rub для получения ответа. Использование других валют требуется прописывать в коде, иначе код будет завершен с ошибкой.
